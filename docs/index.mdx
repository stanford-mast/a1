# Welcome to A1

A1 is an agent compiler that transforms natural language agent descriptions into executable, type-safe Python code. Build reliable AI agents with ahead-of-time (AOT) or just-in-time (JIT) compilation.

## Quick Start

Install A1 and build your first agent in minutes:

```bash
pip install a1
```

```python
from a1 import Agent, Tool, Runtime, LLM

# Define a calculator tool
@tool
def calculator(a: int, b: int, operation: str) -> int:
    """Perform basic arithmetic operations."""
    if operation == "add":
        return a + b
    elif operation == "multiply":
        return a * b

# Create an agent
agent = Agent(
    name="math_agent",
    description="An agent that solves math problems",
    tools=[calculator, LLM("gpt-4o-mini")]
)

# Execute with JIT
async with Runtime() as runtime:
    result = await runtime.jit(agent, query="What is 5 times 6?")
    print(result)
```
