A1 is a new kind of agent framework. It takes an `Agent` (a set of tools and a description) and compiles either AOT (ahead-of-time) into a `Tool` or JIT (just-in-time) for immediate execution optimized for each unique agent input.

## Why use an agent compiler?

1. **Safety** - Generates code for each unique agent input, shrinking the prompt injection attack surface
2. **Speed** - Makes codegen practical with aggressive parallelism and static checking
3. **Determinism** - Optimizes via an engineered cost function (e.g., replacing LLM calls with RegEx)
4. **Flexibility** - Tools from OpenAPI, MCP servers, DB connections, fsspec paths, Python functions, or doc URLs

## Quick Example

```bash
pip install a1-compiler
```

```python
from a1 import Agent, tool, LLM
from pydantic import BaseModel

@tool
async def add(a: int, b: int) -> int:
    return a + b

class MathInput(BaseModel):
    problem: str

class MathOutput(BaseModel):
    answer: int

agent = Agent(
    name="math_agent",
    description="Solves simple math problems",
    input_schema=MathInput,
    output_schema=MathOutput,
    tools=[add, LLM("gpt-4o")],
)

# Compile ahead-of-time
compiled = await agent.aot()
result = await compiled.execute(problem="What is 2 + 2?")

# Or execute just-in-time
result = await agent.jit(problem="What is 5 + 3?")
```
