## Python Functions

Create tools from any async function:

```python
from a1 import tool

@tool
async def search(query: str) -> list[str]:
    """Search the web."""
    return results

@tool
async def calculate(expression: str) -> float:
    """Evaluate math expression."""
    return eval(expression)
```

Supports primitive and complex return types (Pydantic models, lists, dicts).

## OpenAPI

Create tools from OpenAPI specifications:

```python
from a1 import openapi

tools = openapi.from_spec("https://api.example.com/openapi.json")
```

## Model Context Protocol (MCP)

Create tools from MCP servers:

```python
from a1 import mcp

tools = await mcp.from_server("path/to/mcp/server")
```

## LLM

LLMs are tools that call language models:

```python
from a1 import LLM

llm = LLM("gpt-4.1")
```

Use in agents like any other tool.

## ToolSet

Group related tools together to organize hierarchically:

```python
from a1 import ToolSet

math_tools = ToolSet(
    name="math",
    description="Mathematical operations",
    tools=[add, multiply, divide]
)

agent = Agent(
    tools=[math_tools, LLM("gpt-4.1")]
)
```

## Done

Built-in `is_terminal=True` tool for marking workflow completion:

```python
from a1 import Done

done = Done()  # Agent can call this to signal completion
```
